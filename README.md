Choco World 🍫
Welcome to Choco World – your ultimate destination for indulging in the rich and delightful world of chocolates! This web application is built with Flask, HTML, and CSS to provide an elegant and seamless user experience for chocolate enthusiasts.

About the Project
Choco World is a visually appealing and user-friendly website designed to showcase a variety of chocolates, share details about their origins, and allow users to explore the magic of chocolate in an engaging way. From premium dark chocolates to creamy milk chocolates, this website is a haven for chocolate lovers!

Features
Home Page: A beautifully designed landing page with an overview of the website.
Chocolate Collection: Explore a curated collection of chocolates with detailed descriptions.
About Us: Learn more about the story and vision behind Choco World.
Responsive Design: Fully responsive and works seamlessly across devices.
Interactive UI: Built with custom HTML and CSS for a polished and interactive user experience.
Technologies Used
Backend
Flask: A lightweight and powerful Python web framework for building the backend logic and serving content.
Frontend
HTML: For structuring the content on the pages.
CSS: For styling and enhancing the visual appeal of the website.
